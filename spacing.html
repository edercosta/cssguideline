Valores para cada espaço

Exemplo código:

{% highlight sass %} $space-xxs: 2px $space-xs: 4px $space-s: 8px $space-m: 16px $space-st: 24px $space-l: 32px $space-xl: 40px $space-xxl: 48px $space-j: 64px {% endhighlight %}

Como usar:

{% highlight sass %} .bloco margin: $space-m $space-s {% endhighlight %}
